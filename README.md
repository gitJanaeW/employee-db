# employee-db
  
## Description

Using MySQL database queries, create, update and delete a employees in your database from the command line. Employees in the database are organized by position, department and role.

## Technologoies
- Node.js
- MySQL
- Inquirer
- console.table

## Installation/Tests

To install, clone the file. Make sure npm is installed and initialized. To test, write "node .\server.js" install into the command line.
